# Experiment 1: Building and Training an Artificial Neural Network (ANN) for Regression and Classification

This repository contains the implementation of **Artificial Neural Networks (ANNs)** using **TensorFlow** and **keras** for both **Regression** and **Binary Classification** tasks as part of a Deep Learning laboratory experiment.

---

## 📌 Objective

The objectives of this experiment are to:

- Understand the architecture of an Artificial Neural Network (ANN).
- Build an ANN for a regression problem.
- Build an ANN for a binary classification problem.
- Train and evaluate neural network models.
- Compare evaluation metrics used in regression and classification.

---

## 📚 Datasets Used

### Part A – Regression

**Dataset:** California Housing Dataset

- Source: Scikit-learn
- Samples: 20,640
- Features: 8
- Target: Median House Value

---

### Part B – Classification

**Dataset:** Breast Cancer Wisconsin Dataset

- Source: Scikit-learn
- Samples: 569
- Features: 30
- Classes:
  - Malignant
  - Benign
